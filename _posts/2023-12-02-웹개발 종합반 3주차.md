# [GPT] 웹개발 종합반 3주차

##  Python

## Python의 Flask 서버
: 웹이 동작하는 서버

## IF
: 조건문(조건이 참(True), 거짓(False) 중 하나를 만족할 때 실행되는 파이썬 문법)
```python
if 조건1:
    # 조건1이 참일 때 실행할 코드
elif 조건2:
    # 조건1이 거짓이고 조건2가 참일 때 실행할 코드
elif 조건3:
    # 조건1과 조건2가 모두 거짓이고 조건3이 참일 때 실행할 코드
else:
    # 위의 모든 조건이 거짓일 때 실행할 코드
```

## FOR
: 반복문(요소의 리스트를 반복하는 파이썬 문법)
```python
a = [1, 2, 3, 4]
# a라는 숫자 값이 담긴 리스트

for fruit in a:
	print(a)
# a라는 리스트의 요소를 돌아가면서 순서대로 반복
```
