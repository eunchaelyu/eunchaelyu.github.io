## 1일 1커밋 (12/15)    
### 프로그래머스 Lv.0 약수의 합      
![image](https://github.com/eunchaelyu/eunchaelyu.github.io/assets/119996957/1ffc6f0d-d1fa-4835-966f-3b4c2be7a498)


```java  
class Solution {
    public int solution(int n) {
        int answer = 0;
        for(int i=1; i<=n; i++){
            if(n % i == 0){
                answer += i;
            }
        }
        return answer;
    }
}
```    

> for문을 사용해서 i가 1부터 n까지 반복한다
> 그 중 n을 i로 나눴을 때 나머지가 0인 경우 n의 약수에 속하므로
> answer에 i를 더해준다
