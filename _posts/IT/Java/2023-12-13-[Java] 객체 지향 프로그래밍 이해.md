## 객체 지향 프로그래밍 이해    
  캡슐화        
: 속성 + 메서드 (올바른 메서드로 접근해야 변화 가능)     

  상속    
: 속성과 메서드를 부모 객체 -> 자식 객체로 물려받아 사용 가능(코드 중복 줄어듦)    

  다형성    
: 다양한 형태로 구현 가능    

  추상화
: 공통된 부분들을 모아 상위개념으로 새롭게 선언
