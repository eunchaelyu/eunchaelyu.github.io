:  temp_html에 forEach문으로 돌린 반복문의 데이터가 들어온다\
아이디값에 위의 내용이 더해짐
```html
<body>
  <div id="아이디값">
  테스트
  </div>

  <script>
        function 클래스명() {
            let fruits = ['변수1','변수2']
            
            $('#아이디값').empty()
            // 기존에 내용들이 지워짐
            fruits.forEach(element => {
                let temp_html = `<p>${element}</p>`
                $('#아이디값').append(temp_html)
                // '아이디값'에 temp_html 더해진다
            });
  </script>
```
